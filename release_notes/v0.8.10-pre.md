
#### FEATURES

- Added a tool/function called `schedule_AI_prompt` that is always available and can be used to schedule an AI prompt for Enkaidu to execute. 
  - The "prompt" is run exactly as if you typed it into Enkaidu. 
  - This mean we can ask the model to execute not only text prompts but also invoke macros and slash commands.
  - This mean by describing how to use a parametrized macro we can have the LLM schedule prompts to execute. 
  - Example: `Look at the src/sucre folder. It defines various libraries. For each of the folders, schedule an AI prompt to invoke a macro to explain the folder which has the following syntax: '!explain_folder path={FOLDERPATH}'. Use this macro invocation syntax exactly.`
