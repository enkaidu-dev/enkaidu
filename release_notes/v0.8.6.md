
#### FEATURES

- Optionally hide the prompt when using a custom prompt. E.g. `/prompt use echo show=no`
- Support multiple named session stacks, with the default (initial) one called `prime`
- Add `/session stack ...` with sub-commands to list (`ls`) sessions stacks, create (`new`) named session stacks, and switch between (`goto`) active session stacks
- Show the current session stack's name as the prefix of the user input prompt. E.g. `@prime > `
