
#### FEATURES

- Optionally hide the prompt when using a custom prompt. E.g. `/prompt use echo show=no`
- Support multiple named sessions, with the default (initial) one called `prime`
- Add `/session ...` sub-commands to list (`ls`) active named sessions, create (`new`) named sessions, and switch between (`goto`) active sessions
- Specify a model name when creating a new session if you want, allowing different sessions to use different LLMs
- Show the current session stack's name as the prefix of the user input prompt. E.g. `@prime > `
