
#### IMPROVEMENTS

- Modified the `read_text_file` tool to accept optional `line_numbers` and `line_range` parameters:
  - `line_numbers`, if `true`, will include line numbers just as `cat -n` does.
  - `line_range`, specified as an array of start- and end-line numbers (1-index), lets the tool return a portion of the file.
- Renamed search-and-replace tool to `str_replace_in_text_file` and improved parameter names.
  - Modified tool to perform one replacement by default
  - Added `multiple` parameter which can be set to` true` to ask the tool make multiple replacements
  - Returns the number of replacements with the modified contents
- Improved description and parameters for all the `FileManagement` tools.
- Fixed `rename_or_move_file` tool to prevent changes to deleted files

#### FEATURES

- Add `str_insert_in_text_file` tool that can be used to insert text at a specific line number in a text file.
  - See improvements to `read_text_file` for some rationale

#### FIXES

- Fixed console rendering in streaming mode to start new lines between different kinds of content.
